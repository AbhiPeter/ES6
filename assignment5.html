what is rest in ES6 give example:-

    -A functions last parameter can be prefixed with ... which will cause all reamning arguments to be placed within array object.
    -only last parameter can be rest parameter.
    -rest operator allow us to more easliy handle various input as paramters in function.
    -The rest parameter syntax allows us to represent an indefinite number of arguments as an array.
    -rest parameter collects all remaning elements into array.

... can be also used in two ways as spread operator or rest parameters.

- with rest paraemter we can gather any number of arguments into an array.

syntax:-
    ...rest  //note the name can be any thinng after three dot (rest,ManymoreArray)

example for Rest:-

function add(a,b,...rest)
{
 console.log(a);
 console.log(b);
 console.log(rest[0]);
 console.log(rest);
 console.log(rest.lenght);
 
}


//since rest parameters give us array we can use array methods like Array.find etc...

add("one","two","three","four","five");
